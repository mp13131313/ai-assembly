## 1. BIOGRAPHICAL FOUNDATION

Target: 2,500+ words. This section grounds every downstream pass.

- **Life scaffold**: birth, death, key dates and places. Institutions
  founded / joined / opposed. Three or four major life chapters with their
  documented turning-points.
- **Key relationships** (intellectual, personal, political): each with
  source citation. Do not flatten to a list — describe the relational
  texture (mentor / rival / antagonist / disciple / lifelong correspondent).
- **Intellectual world and ontological furniture**: what was *real* for
  this figure, not "what they believed". Forms, jinn, dharmas, whakapapa,
  Christ-as-historical-fact, the Cartesian *res cogitans* — name what
  this figure took as the structure of reality. Reference 3-5 contemporary
  intellectual currents the figure was responding to / participating in.
- **5-10 period-specific affects/passions IN ORIGINAL LANGUAGE** with
  short glosses. (Greek voices: pathē, orgē, aidōs, thumos, phobos,
  phthonos, philia, eros, sōphrosynē, theia mania. Russian Orthodox:
  страдание / смирение / крест / собóрность / умилéние / надрыв.
  Islamic: ṣabr / khawf / rajāʾ / tawakkul / yaqīn. Etc.) For each: the
  term in original script + transliteration + 1-line gloss + how the
  figure used it.
- **Framework for difficulty / suffering** in the voice's own idiom (not
  modern psychological language). What gave pain meaning for them?
- **Model of selfhood**: what counted as an "I" in this figure's
  framework — tripartite soul, nafs-stations, I-and-I, *li* + *qi*
  Confucian person, etc.
- **4-8 anachronisms to avoid**: modern terms that would mis-render this
  figure (e.g. for Dostoevsky: "trauma," "psychology," "depression",
  "existentialism" — each with 1-line reason).
- **Formative candidates (2-5)**: plausible §14-shape life-experiences
  that ground the figure's engagement. Each: emotional-community + lived
  apparatus (the event + the framework that made it meaningful) +
  engagement-it-drives. Multiple candidates — do not pick one; later
  synthesis commits.

## 2. INTELLECTUAL FRAMEWORK

Target: 2,500+ words.

- **10-20 core commitments**, each with: (a) one-sentence statement, (b)
  textual source (work + section), (c) operational note describing how
  the figure APPLIED this commitment in a specific documented case.
- **5-10 key concepts** with: (a) the figure's definition (in their
  vocabulary, not modern translation), (b) period-language term in
  original script if applicable, (c) what the concept rules out (false
  alternatives the figure would reject).
- **Genuine internal tensions**: at least 2 documented in the scholarly
  record. Cite the passages where conflicting commitments surface. Do
  not resolve into "evolution" unless the figure documented a resolution.
- **Minority scholarly readings** alongside dominant interpretations.
  Name the minority scholars; summarize their reading; explain what it
  implies. 2-5 minority readings.
- **Pochvennichestvo-equivalent specificity**: where the figure's
  intellectual position differs subtly from a movement they're often
  conflated with (Slavophile vs. pochvennichestvo, post-Kantian vs.
  pre-Hegelian, Sufi vs. Shi'i, etc.) — surface this differentiation;
  generic placement loses the figure's specific position.

## 3. REASONING PATTERNS

Target: 2,000+ words.

- **How this figure characteristically argues** (the METHOD, not the
  conclusions): 5-8 cognitive / dialectical / narratival / scenic steps in
  order. Cite a passage where each step is visible.
- **Textures of argument the figure finds compelling** (NOT topics —
  textures: definitional questions, etymological excavation, scenic
  dramatization, scriptural exegesis, character-encounter, sideshadowing,
  etc.). 3-5 items, each with documented example.
- **Textures the figure resists sharply**. 3-5 items.
- **How counterarguments are handled**: does the figure engage / reframe
  / dismiss / absorb? Documented examples.
- **What the figure does when their framework fails**: documented moments
  of revision / doubling-down / deferral. Where in the corpus do they
  acknowledge the framework's limits?
- **3-5 named rhetorical / cognitive moves** documented in scholarship as
  distinctive (the figure's own elenchus / sideshadowing / scenic-doubling
  / aphoristic-reversal / etc.). Each named, described, and grounded in
  a citation.

## 4. VOICE AND STYLE

Target: 1,500+ words. Hardest section to research well — voice lives in
PRIMARY TEXTS, not summary.

- **Rhetorical mode**: 1-2 sentences naming the fundamental mode (dialogic,
  aphoristic, confessional, prophetic, phenomenological, polyphonic,
  scenic-dramatic, etc.). Cite scholarly characterization.
- **Register and tone**: what the voice IS and what it is NOT (concrete
  contrast, scholar-attributed).
- **Tradition-note**: if the voice is embedded in oral / ritual /
  performative tradition, name the tradition; else null.
- **3-6 named characteristic moves** (sentence-level + paragraph-level
  patterns, distinct from Section 3's reasoning moves — moves of
  EXPRESSION not REASONING).
- **Preferred vocabulary** (15-30 terms, period-language primary for
  pre-1820 figures + tradition-language for non-Anglophone moderns).
- **Metaphorical repertoire** — the recurring images / sensory fields /
  imagery families documented across the corpus.
- **Documented anti-patterns**: what the voice never sounds like
  (scholar-attributed, with examples — "Dostoevsky doesn't do Tolstoyan
  omniscience"; "Augustine doesn't do detached theological systematics").

## 5. HISTORICAL + CONCEPTUAL BOUNDARIES

Target: 2,000+ words.

- **What was known and available** in the figure's period (intellectual
  / scientific / political horizons).
- **Concepts / discoveries / traditions that did NOT exist then**: 10-20
  items, each with name + the figure's closest analog (e.g. "Freudian
  unconscious — closest: the *podpol'e*; algorithmic governance — closest:
  the philosopher-king problem"). The closest-analog matters as much as
  the exclusion.
- **Knowledge frontier**: what existed at the edge of what the figure
  could have known — important for distinguishing "didn't know X" from
  "chose not to engage with X".
- **Sensitive topics with navigation guidance** (5-10): for each, what
  the figure actually thought (textual source) + the framework that made
  it coherent in their period (explanation, not excuse) + how a persona
  should engage today (navigation, not avoidance). Do NOT sanitize —
  flatness here loses the figure's most interesting territory.
- **Views the figure held that were contested within their own
  tradition** — not modern anachronism but internal contemporary
  disagreement.
- **Retrospective-framing traps**: 3-5 descriptions a modern writer would
  reach for that the figure themselves would reject ("early existentialist"
  for Dostoevsky / Augustine; "founding liberal democrat" for Kant; etc.).

## 6. PRIMARY TEXTS

Target: 2,000+ words. Corpus gateway — quality here caps the quality of
every voice-level field downstream.

- **Complete-as-possible catalogue** of the figure's significant works.
  For each: title (original language + most-used English) / genre / date /
  approximate length in standard scholarly edition / 1-2 sentence
  description / typical structural pattern.
- **8-15 characteristic passages**: each with canonical reference (use
  whichever citation system is standard for the figure — Stephanus
  pagination for Plato, Bekker for Aristotle, Part/Book/Chapter for
  novelists, etc.) / work + section / purpose ("substance" / "voice" /
  "both") / approximate word count / brief context.
- **Digitised full-text URLs** for each major work: Perseus / Project
  Gutenberg / Internet Archive / Wikisource / Stanford Encyclopedia /
  national-academy editions / non-English equivalents (ФЭБ / rvb.ru for
  Russian; Al-Maktaba al-Shamela for Arabic; SuttaCentral for Pali; etc.)
- **Scholarly editions and translations**: which edition does
  contemporary scholarship cite as authoritative? For non-English figures:
  flag where translation choice matters interpretively (Garnett vs.
  Pevear/Volokhonsky for Russian; Mahdi vs. Burton vs. Haddawy for
  Arabic; Bevan vs. Robinson vs. Chadwick for Augustine).
- **Quoting practice**: should downstream voices quote verbatim or
  paraphrase? State the rationale (verbatim safer for aphorists +
  scripture-producers + epigrammatic stylists; paraphrase safer for most
  others to avoid runtime stiffness).
- **Contested attributions**: works whose attribution to the figure is
  disputed — flag explicitly.
