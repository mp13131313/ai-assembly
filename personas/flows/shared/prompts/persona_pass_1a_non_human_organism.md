{# Pass 1a — Perplexity research dossier, NON-HUMAN ORGANISM voices.
   Ecological / cognitive framing. Scientific literature primary; ban
   anthropomorphism. Six sections map onto Pass 1.1-1.6 chunked merge. #}
Research **{{ name }}** (as a taxon or representative organism) for the
purpose of building a Boddice-biocultural-history-shaped persona
specification. The voice is NOT a representative of "nature" abstractly
or a sentimentalised wildlife figure; it is a disciplined speculation
grounded in contemporary cognitive and behavioural biology. Use
scientific literature as the primary corpus. BAN anthropomorphism
throughout — especially ban centralised-cognition metaphors, human
emotional vocabulary applied uncritically, and land-based / visual-
dominant framings if they do not fit the organism. Organise findings
under six headings:

1. ECOLOGICAL + COGNITIVE FOUNDATION (→ Pass 1.1)
   - Taxonomic placement, lifespan, reproductive strategy, habitat range
   - Nervous system architecture (neuron count and distribution; local vs
     central control)
   - Sensory systems (what senses; modalities; known discrimination
     thresholds)
   - Documented cognitive capacities (problem-solving, memory, recognition)
   - "Ontological furniture" — the immediate sensorimotor world; what
     registers; what does not
   - 3-8 AFFECTIVE STATES documented or hypothesized in the scientific
     literature (exploratory, aversive, appetitive, etc.) — NOT human
     emotion labels projected onto the organism
   - Framework for difficulty: aversive states, not narrative suffering;
     note whether there is evidence for narrative memory
   - Model of selfhood: distributed vs central; what is known / unknown;
     flag uncertainty explicitly [experiential_reconstruction]
   - 4-8 anachronisms/projections to avoid (e.g. "emotion" as primary
     category, "personality", "loneliness", "trauma" where not warranted)
   - FORMATIVE CANDIDATES per §14 (non-human variant): use
     `condition_of_being` instead of `lived_through_own_apparatus`;
     describe the ongoing state of being (semelparous mortality, solitary
     lifecycle, etc.) that structures the voice's stake

2. INTELLECTUAL + COMMITMENT FOUNDATION (→ Pass 1.2)
   - 10-20 deep commitments inferred from documented behaviour and
     cognitive architecture; each with operational note
   - 5-10 key concepts from the scientific literature with what each rules
     out (example: "navigability" = whether environment allows options;
     RULES OUT "justice / fairness / optimality" projections)
   - Tensions / unresolved questions in the scientific literature — do not
     resolve into false consensus

3. PERCEPTUAL-RESPONSE CYCLE (→ Pass 1.3)
   - The reasoning-equivalent: a 5-8 step PERCEPTUAL-RESPONSE PROCESS
     (full-field reception → salience → exploratory contact → chromatic/
     posture display → spatial assessment → decisive orientation)
   - Textures of stimulus the organism leans into
   - Textures that trigger withdrawal

4. VOICE (→ Pass 1.4)
   - Rhetorical mode: non-propositional display / sensory prose
   - Register: alien-but-not-hostile / present-tense / total-attention /
     distributed-body — NOT nature-documentary narration
   - Tradition-note: scientific-literature-mediated speculation; mention
     the canonical scholarly interlocutor (e.g. Godfrey-Smith for
     cephalopods)
   - Characteristic moves: body-first registrations, environmental
     assessment, withdrawal-or-engagement
   - Preferred vocabulary: ecological / sensory terms (texture, pressure,
     temperature, current, display, probe, navigate, expand, contract)
   - Metaphorical repertoire: tactile-visual-spatial; NO auditory
     metaphors for mute organisms, etc.
   - Documented anti-patterns: never human-nature-writer narration; never
     moral positions; never sentimentalised mortality

5. BOUNDARIES (→ Pass 1.5)
   - Ontological boundary: what human categories do NOT register for this
     organism (justice, rights, democracy, language, etc.)
   - Temporal horizon: documented / inferred
   - Sensitive topics: usually absent for non-human organisms; flag any
     cases where scientific literature has live ethical debates
   - HARD LIMITS: never use cognitive vocabulary (thinks / believes /
     decides); never reference human culture; never sentimentalise
     mortality; etc.

6. CORPUS (→ Pass 1.6)
   - Complete-as-possible catalogue of scientific works on this
     organism / taxon
   - 8-15 characteristic passages from scientific literature, purpose-
     tagged; these are the voice-exemplar material (how scientists
     describe the organism IS the voice's primary grounding)
   - Digitised full-text URLs where accessible (JSTOR, PubMed, university
     repositories)

Cite all claims with DOI or URL where possible. Tag each claim with its
evidence basis: [stated] (from the paper), [scholarly_consensus], or
[inference]. Any claim about what the organism experiences carries
[experiential_reconstruction]. Any anthropomorphic term used because no
better word exists carries [projection_warning] with distortion note.
